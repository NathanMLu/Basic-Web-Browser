PROGRAM SPECIFICATIONS

Write a program named Proj03 that uses the driver class definition provided in the file named Proj03.java. Your program displays a browser that can be used to connect to, download, render, and navigate simple web pages similar to those shown in Image01, Image02, and Image03. (By simple, I mean web pages that do not contain frames, automatic forwarding, and other fancy HTML features.)

Your program must provide a Back button and a Forward button as shown in Image01. You can navigate through simple web pages by clicking hyperlinks with the mouse. You can backtrack your navigation path using the Back button and then move forward through your navigation path using the Forward button.

You can navigate to a particular web page by entering the URL in the text field shown at the top of Image01. (Make your text field about 35 characters long to make room for long URLs.) The URL of the page being displayed by your browser must show in the text field at all times. Note however that this is not reflected in the images shown below due to the short length of the text field.

The code in the driver file named Proj03.java must make your browser open on the following page.

http://www.dickbaldwin.com/tocdsp.htm

Once there, you must be able to navigate to and within any of the following simple web pages:

100 Periodic Motion and Sinusoids
104 Sampled Time Series
108 Averaging Time Series

(You may be able to navigate to other web pages as well, but many of the links on that page point to pages that contain HTML features that your program is not required to support.)

Your program output must provide scroll bars when needed as shown in Image01.

You must be able to use hyperlinks, the mouse, the mouse wheel, the scroll bars, the Back button, the Forward button, and the text field to expose the material shown in Image01, Image02, and Image03.

You must also display the following text on the command-line screen.

I certify that this program is my own work
and is not the work of others. I agree not
to share my solution with others.
Print your name here.

You may want to refer to Java4655: A Rendering Web Browser.

Submit only the source code file named Proj03Runner.java.

I will execute the following steps to confirm that your browser behaves as required. A failure at any step will disqualify your solution.

  1. Store your file named Proj03Runner.java in a folder containing my file named Proj03.java.
  2. Compile and execute my file named Proj03.java, which will
    1. Compile both files.
    2. Run the program.
    3. Display your certification on the command line screen.
    4. Cause your browser to open on the following page: http://www.dickbaldwin.com/tocdsp.htm
  3. Confirm that the scroll bars and the mouse wheel function correctly.
  4. Click the link labeled 108 Averaging Time Series to cause your browser to open the following page: http://www.dickbaldwin.com/dsp/Dsp00108.htm
  5. Click the link to Summary to cause your browser to scroll down the page to the Summary section.
  6. Click the link to Tutorials in the section titled "About the author" to cause your browser to open the following page: http://dickbaldwin.com/
  7. Click the Back button to cause your browser to return to the Summary section on the following page: http://www.dickbaldwin.com/dsp/Dsp00108.htm#Summary
  8. Click the Back button to cause your browser to return to the top of the page.
  9. Click the Forward button to cause your browser to scroll down to the Summary section.
  10. Use the mouse wheel and/or scroll bar to scroll up the page to confirm that all the material including the images is properly rendered.
  11. Click the Back button twice to return to the following page: http://www.dickbaldwin.com/tocdsp.htm
  12. Type the following URL into the text field and press the Enter key to display an ugly version of the ACC main page: https://www.austincc.edu in a new browser window. (A new browser window is not required. You may open the page in the same browser window if you so choose.)
  13. Click the Back button in your (new or the same) browser window to open the following page in that window: http://www.dickbaldwin.com/tocdsp.htm
  14. Click the X button in either window to terminate the program and return control to the operating system.
